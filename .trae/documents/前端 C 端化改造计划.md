# 前端界面 C 端化改造计划

## 1. 视觉与交互重构 (UI/UX)
### 目标：打造沉浸式、极简的聊天体验
- **CSS 深度定制**：
  - 隐藏 Streamlit 默认的顶部汉堡菜单、Footer 和右上角状态栏，减少干扰。
  - 调整页面边距，使聊天区域更聚焦。
  - 优化字体和配色，使用更温暖、亲和的色调。
- **主界面净化**：
  - **移除** 顶部的 "AI育儿助手 Demo" 大标题，改用更精致的页面布局。
  - **移除** 蓝色的 "欢迎使用..." 提示框。
  - **新增** 智能欢迎语：当对话为空时，AI 自动发送一条友好的开场白（如“你好！我是你的育儿助手，有什么可以帮你的吗？”），而不是显示静态提示。

## 2. 内容精简 (Content Cleanup)
### 目标：删除开发者导向内容，保留核心功能
- **侧边栏 (Sidebar)**：
  - **删除** "使用说明"、"测试场景"、"系统状态" 等大段文字。
  - **保留** 核心功能按钮："🗑️ 开启新对话"。
  - **保留** 必要的免责声明（以小字形式展示在侧边栏底部）。
- **聊天区域**：
  - **删除** "📊 技术细节" 折叠面板（C端用户不需要看到 JSON 数据）。
  - **优化** 错误提示：将 "API错误" 等技术报错转化为更友好的提示（如“网络有点小差错，请稍后再试”）。

## 3. 文件修改清单
- `frontend/streamlit_app.py`:
  - 注入自定义 CSS 样式块。
  - 重写侧边栏布局逻辑。
  - 移除技术细节展示代码。
  - 优化 Session State 初始化逻辑（加入默认欢迎语）。

## 效果预期
改造后，界面将类似微信或 ChatGPT 的简洁风格：左侧仅保留必要操作，右侧为纯净的对话流，不仅视觉上更美观，也更符合普通家长的使用习惯。
