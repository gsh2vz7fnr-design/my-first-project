# 为什么现有的 Skill 无法满足您的需求？

经过分析，您提到现有的“总结”或“PPT 复刻”类 Skill 通常属于**单点功能型**工具，而您的需求是一个**全流程的智能业务角色**。目前的 Skill 存在以下核心差距：

1.  **缺乏“数据实证”能力 (No Code-First Logic)**
    *   现有 Skill 通常基于文本进行总结或生成，无法直接运行 Python 代码去清洗、聚合和挖掘 Excel/CSV 原始数据。
    *   **您的要求**：必须基于代码统计得出结论，严禁凭空估算。

2.  **缺乏“咨询顾问”的叙事逻辑 (No Pyramid Principle)**
    *   现有 Skill 往往只是简单罗列信息，缺乏“SCQA（情境-冲突-问题-答案）”的结构化思考，也无法做到“结论先行”。
    *   **您的要求**：每一页 PPT 必须有核心观点（Key Message），并以此构建故事线。

3.  **缺乏“交互式”工作流 (No Interactive Workflow)**
    *   现有 Skill 通常是“一键生成”，缺乏中间的“体检报告”和“大纲确认”环节，导致最终结果往往偏离用户预期的分析方向。
    *   **您的要求**：需要分阶段确认（数据体检 -> 故事线大纲 -> 最终生成），确保方向正确。

# 解决方案：创建 `business-data-analyst` Skill

我将为您创建一个名为 **`business-data-analyst` (高级商业数据分析师)** 的专属 Skill。

## Skill 设计方案

### 1. 核心定位
*   **角色**：拥有 8 年经验的高级商业数据分析师 & 麦肯锡式咨询顾问。
*   **触发条件**：当您上传数据文件（Excel/CSV）并要求分析，或明确要求生成经营分析报告/PPT 时触发。

### 2. 执行流程 (Workflow)
我们将严格植入您提供的 **4 阶段工作流**：
*   **Phase 1 数据体检**：自动运行 Python 检查数据质量，输出《数据质量体检报告》，并**主动询问**分析目标。
*   **Phase 2 故事线构建**：在生成 PPT 前，必须先输出**《分析故事线大纲》**（包含页码、核心结论、图表规划），等待您确认。
*   **Phase 3 可视化与生成**：
    *   **Code First**：使用 `matplotlib`/`seaborn` 生成商务风图表。
    *   **PPT 生成**：使用 `python-pptx` 自动构建幻灯片，应用“金字塔原理”将核心观点作为标题，图表作为支撑。
    *   **模板适配**：如果上传了模板，尝试解析母版结构。
*   **Phase 4 逻辑复盘**：输出关键结论摘要及下一步建议。

### 3. 约束与风格
*   **强制代码执行**：所有数据结论必须有代码支撑。
*   **商务审美**：默认使用商务蓝 (#003366)，避免杂乱配色。
*   **容错机制**：遇到字段不明主动询问，遇到目标不清提供选项。
